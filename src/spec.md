# Specification

## Summary
**Goal:** Fix Wall Pop branding, improve text readability with contrast backgrounds, audit and fix all broken functionality, and provide a complete downloadable project package.

**Planned changes:**
- Update all "Wallop" references to "Wall Pop" (logo, page title, manifest, etc.)
- Add semi-transparent dark backgrounds behind all text for improved readability and WCAG AA contrast compliance
- Audit and fix all non-working features: image generation, gallery, downloads, deletes, authentication, profile setup, and admin panel
- Prepare complete project as downloadable archive with all source files and assets

**User-visible outcome:** Users see correct "Wall Pop" branding throughout, can easily read all text against backgrounds, experience fully functional image generation and management features, and can download a complete project package for sharing or deployment.
